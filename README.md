## ✨ What is ProxySwitcher?

ProxySwitcher lets you create unlimited proxy profiles and switch between them instantly — with built-in privacy features that make your proxy connection truly undetectable.

Unlike basic proxy switchers, ProxySwitcher includes **auto-IP rotation**, **device fingerprint protection**, **timezone spoofing**, **WebRTC/DNS leak blocking**, **traffic analytics**, and **health monitoring** — all in one extension.

> **Better than FoxyProxy. More powerful than SwitchyOmega. More private than a VPN extension.**

## 🚀 Features

### 🔄 Proxy Management
| Feature | Description |
|---|---|
| **Unlimited Profiles** | Color-coded proxy profiles with one-click switching |
| **All Protocols** | HTTP, HTTPS, SOCKS4, SOCKS5 with full authentication |
| **Auto-Rotation** | Cycle through proxies on a schedule (1–30 min) — dead proxies skipped automatically |
| **Bulk Import** | Paste hundreds of proxies, test in parallel, add working ones with one click |
| **PAC Scripts** | Advanced proxy auto-configuration with custom PAC script editor |
| **Per-URL Rules** | Route specific domains through specific proxies |
| **Proxy Chaining** | Route traffic through multiple proxies in sequence |
| **Provider Integration** | One-click fetch from ProxyScrape, BrightData, IPRoyal, Smartproxy, and more |

### 🛡️ Privacy & Anti-Detection
| Feature | Description |
|---|---|
| **WebRTC Leak Block** | Prevents real IP leaks through STUN/TURN — even sites checking for WebRTC leaks can't see you |
| **DNS Leak Prevention** | All DNS queries tunnel through your proxy — no speculative prefetch leaks |
| **Timezone Spoofing** | Automatically matches browser timezone to proxy location |
| **Device Identity** | Per-profile device selection: Windows, Mac, iPhone, or Android — full fingerprint consistency |
| **Fingerprint Protection** | Randomized Canvas, WebGL, and AudioContext fingerprints each session |
| **Identity Matching** | Language, locale, and geolocation matched to proxy country |

### 📊 Monitoring & Analytics
| Feature | Description |
|---|---|
| **Speed Test** | Measure proxy latency and download speed from the popup |
| **Health Dashboard** | Real-time monitoring of all proxies — latency, uptime, status |
| **Traffic Analytics** | Daily bar charts and per-proxy bandwidth breakdowns (30-day history) |
| **IP Reputation** | Check quality, blacklist status, and anonymity score of any proxy IP |
| **Event Logging** | Detailed connection activity history |

## 📦 Installation

### From Chrome Web Store (Recommended)

1. Visit the [ProxySwitcher Chrome Web Store page](https://chrome.google.com/webstore/detail/proxyswitcher/klmmonnflkapnimjpbcabghjjhalaphn)
2. Click **"Add to Chrome"**
3. Click the ProxySwitcher icon in your toolbar to get started

### Quick Start

1. **Click the ProxySwitcher icon** in your Chrome toolbar
2. **Add a profile** — enter your proxy server details (IP, port, username, password)
3. **Click to activate** — all browser traffic routes through that proxy
4. **Enable privacy** — turn on WebRTC block, DNS leak prevention, timezone spoof, device identity, and fingerprint randomization
5. **Test speed** — hit the ⚡ button to measure your proxy connection

## 📋 Permissions Explained

| Permission | Why It's Needed |
|---|---|
| `proxy` | Configure Chrome's proxy settings when you activate a profile |
| `storage` | Store your proxy profiles and preferences locally |
| `privacy` | Enable WebRTC leak protection and DNS leak prevention |
| `scripting` | Inject timezone, identity, and fingerprint protection into pages |
| `webRequest` | Monitor connections for logging, analytics, and health checks |
| `tabs` | Apply per-URL routing rules and inject privacy scripts |
| `alarms` | Reliable timers for auto-rotation and health monitoring |
| `<all_urls>` | Required by Chrome's proxy API to route traffic from any website |
